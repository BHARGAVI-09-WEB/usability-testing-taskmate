# Usability Testing & Analysis – TaskMate

## 📱 Project Overview
This project focuses on evaluating the usability of a student task manager app through testing and feedback.

## 🎯 Objectives
- Identify usability issues
- Improve user experience
- Analyze user interaction

## 🧪 Testing Process
- Conducted usability testing with sample users
- Observed user behavior
- Collected feedback

## 📊 Results
- Simple and easy navigation
- Minor issues in button visibility
- Suggested improvements applied

## 🛠 Tools Used
- Figma
- Microsoft Word
